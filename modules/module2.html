Модуль 2: Python — База для Автоматизатора

Теорія — це важливо, але справжня магія починається з коду. У цьому модулі ми вивчимо "абетку" мови Python — ті основні інструменти, які потрібні кожному автоматизатору щодня.

1. Змінні та Типи Даних

Змінна — це як підписана коробочка, в яку ми можемо покласти якусь інформацію. Основні типи інформації (даних), які ми використовуємо:

  • Рядок (string, str): Будь-який текст у лапках. Наприклад: "Привіт, світ!".
  • Ціле число (integer, int): Числа без дробової частини. Наприклад: 101.
  • Булевий тип (boolean, bool): Дуже важливий для тестів! Має лише два значення: True (правда) або False (хибність). 

# Створюємо змінні
test_name = "Перевірка логіну"
test_id = 5
is_passed = True

2. Умови (if / else)

Дозволяють нашій програмі приймати рішення. "Якщо умова правдива — роби це, **інакше** — роби щось інше". 


if is_passed == True:
    print("Тест успішний! ✅")
else:
    print("Тест провалено! ❌")

3. Цикли (for)

Дозволяють повторювати одну й ту саму дію багато разів. Наприклад, перевірити кожен елемент у списку. 


# Створюємо список (list)
browsers = ["Chrome", "Firefox", "Safari"]

# Цикл, що пройдеться по кожному елементу
for browser in browsers:
    print(f"Запускаю тест у браузері {browser}...")

4. Функції (def)

Це як створити власний інструмент. Ми "упаковуємо" шматок коду, даємо йому ім'я і можемо потім викликати його багато разів. Це слідує принципу DRY (Don't Repeat Yourself - Не повторюй себе). 


# Оголошуємо функцію
def print_separator():
    print("---===---")

# Викликаємо функцію
print("Початок тесту")
print_separator()
print("Кінець тесту")

Інтерактивний Тест до Модуля 2

Що буде результатом виконання коду: `x = "5"`, `y = "3"`, `print(x + y)`?

"53"
8
Помилка

Оскільки і "5", і "3" знаходяться в лапках, вони є рядками (strings). Оператор `+` для рядків означає конкатенацію (склеювання), а не математичне додавання.

Який оператор використовується для перевірки рівності двох значень?

=
==
!=

Одинарний знак `=` використовується для присвоєння значення змінній (`x = 5`), тоді як подвійний `==` використовується для порівняння ("чи дорівнює `x` п'яти?").

Що таке `список` (list) в Python?

Незмінний набір елементів.
Впорядкований, змінний набір елементів у квадратних дужках.
Набір унікальних елементів у фігурних дужках.

Списки `[...]` є однією з найпоширеніших структур даних в Python. Вони впорядковані (мають індекси) і їх можна змінювати (додавати, видаляти елементи).

Що означає ключове слово `pass` в Python?

Нічого не робити; це заглушка.
Зупинити виконання програми.
Пропустити поточну ітерацію циклу.

`pass` використовується, коли синтаксично потрібен якийсь код, але вам не потрібно, щоб щось виконувалось. Наприклад, у порожній функції або класі, який ви плануєте написати пізніше.
